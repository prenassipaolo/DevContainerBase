# DevContainerBase
Create a base repo to use the Dev Containers.

The project contains as an example a lightweight, modular Task Management API built with FastAPI. The focus is on the setup of the project as a minimum and flexible starting point for every new project.

## Features
- **CRUD Operations**: Create and Read tasks.
- **Auto-Documentation**: Interactive Swagger UI at `/docs`.
- **Validation**: Strict data validation using Pydantic.
- **Testing**: Automated tests using Pytest.
